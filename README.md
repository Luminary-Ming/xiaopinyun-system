## 校聘云 — 校园招聘综合服务平台

校聘云是一个面向 **学生**、**企业HR** 和 **系统管理员** 的在线招聘与求职管理平台。平台旨在为高校学生提供便捷的简历管理、职位搜索与投递服务，同时为企业HR提供职位发布、简历筛选与人才管理功能，并支持管理员进行系统审核与数据统计。

## 技术栈 

`Spring Cloud Gateway` + `Nacos` + `Vue3` + `Vue Router` + `Element Plus` + `MySQL` + `MinIO`

## 核心功能模块

### 1. 用户角色与权限管理

- **学生角色（role=0）**
  - 注册/登录后完善在线简历（个人信息、求职期望、教育背景、工作经历、项目经历、专业技能等）
  - 浏览校招/实习职位，投递简历，收藏感兴趣职位
  - 附件管理（上传/下载/预览 PDF 简历）
  - 查看投递反馈与HR回复
- **企业HR角色（role=1）**
  - 维护公司信息（工商信息、福利待遇、公司介绍等）
  - 发布/编辑/删除招聘职位
  - 查看收到的简历投递，进行“通过/退回/感兴趣”等筛选操作
  - 回复学生投递，管理感兴趣的人才库
- **管理员角色（role=2）**
  - 审核学生简历信息是否合规
  - 审核企业发布的职位信息
  - 查看用户增长、企业注册、就业率等统计图表
